Forecasting Model: CAGR

We use the Compound Annual Growth Rate (CAGR) to project future earnings.

$$ CAGR = \left( \frac{\text{Ending Value}}{\text{Beginning Value}} \right)^{\frac{1}{\text{n}}} - 1 $$

Where:
Ending Value (EV): The value of the investment at the end of the period.
Beginning Value (BV): The value of the investment at the start of the period.
n: The number of compounding periods (years).

Performance Analysis Script

Python script for calculating rolling averages on stock performance.


import pandas as pd

# Load quarterly performance data
data = pd.read_csv('q3_performance.csv')

# Calculate 30-day rolling average
data['rolling_avg'] = data['stock_price'].rolling(window=30).mean()

# Output the latest figures
print(data[['date', 'stock_price', 'rolling_avg']].tail())
