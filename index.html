Portfolio Risk: Monte Carlo Simulation

Python script snippet for simulating potential portfolio outcomes.


import numpy as np
import pandas as pd

# Define simulation parameters
num_simulations = 10000
num_days = 252 # Trading days in a year

# Fetch historical returns for a portfolio
log_returns = np.log(1 + portfolio_data['daily_return'])
mu = log_returns.mean()
sigma = log_returns.std()

# Run the Monte Carlo simulation
daily_returns = np.exp(mu + sigma * np.random.randn(num_days, num_simulations))
price_paths = np.zeros_like(daily_returns)
price_paths[0] = portfolio_data['price'].iloc[-1]

for t in range(1, num_days):
    price_paths[t] = price_paths[t - 1] * daily_returns[t]

# Calculate risk metrics like Value at Risk (VaR)
VaR_95 = np.percentile(price_paths[-1], 5)
This code demonstrates how we model future uncertainty. We run thousands of simulations to understand the range of potential outcomes for our portfolios. The key part is the final step, where we calculate Value at Risk, giving us a hard number for our potential downside. This is critical for regulatory compliance and internal controls.
